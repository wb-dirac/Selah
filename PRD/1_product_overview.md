## 1. 产品概述

### 1.1 产品定位

一款以**隐私优先、本地运行**为核心理念的跨平台智能个人助理应用。用户可通过自然语言（文字、语音、图片）与助理交互，助理能调用手机系统能力、唤起第三方应用、执行后台任务，并通过 A2A 协议与外部 Agent 服务协作，完成复杂的跨系统任务。

### 1.2 核心价值主张

| 价值点 | 说明 |
|--------|------|
| **隐私本地化** | 所有敏感数据（通讯录、对话历史、API Key）存储在设备本地，加密保护，不上云 |
| **模型自由选择** | 支持本地模型（Ollama/llama.cpp）和任意云端 LLM，用户持有并自管理 API Key |
| **生成式 UI** | 助理响应不局限于纯文字，可动态渲染商品卡片、地图预览、日历事件等结构化组件 |
| **开放 Agent 生态** | 支持 Google A2A 协议，助理可作为 Agent 网络的节点，与其他 Agent 服务协作 |
| **标准 Skill 生态** | 遵循 Anthropic Agent Skill 标准，对接开放 Skill 市场，所有 Skill 脚本运行在安全沙箱中 |
| **开源** | 项目完全开源，社区共建，用户可自由审查代码、部署和贡献 |

### 1.3 产品边界（第一版不包含）

- 多用户 / 团队协作功能
- 云端数据同步（Phase 2 通过 GitHub Gist 实现）
- 图片生成（手机端算力不可行）
- 自训练 / Fine-tune 模型
