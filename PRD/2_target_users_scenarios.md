## 2. 目标用户与使用场景

### 2.1 目标用户画像

**主要用户**：科技从业者、效率工具爱好者、注重隐私的用户，年龄 25-45 岁，有一定技术背景但不要求专业开发能力。

**次要用户**：开发者（Skill 开发、Agent 集成、开源贡献者）。

### 2.2 核心使用场景

**场景一：日常信息查询与任务执行**
> 用户说："帮我查一下明天北京到上海的高铁，推荐早上出发的，订好后加到我的日历"
> 
> 助理通过 A2A 调用铁路查询 Agent → 生成式 UI 展示班次卡片 → 用户确认 → 写入本地日历 → 设置出行提醒

**场景二：多模态信息处理**
> 用户拍一张餐厅菜单照片，问："帮我算一下点这三道菜总共多少钱，热量够不够两个人吃"
> 
> 助理调用本地 OCR 解析图片 → 提取价格和食材信息 → 计算并回答

**场景三：智能提醒与后台任务**
> 用户说："明天下午三点提醒我给张总回邮件，如果我还没回就帮我起草一封"
> 
> 助理注册定时任务 → 到点推送通知 → 检测是否已回复 → 未回复则触发邮件起草

**场景四：Skill 扩展**
> 用户在 Skill 市场搜索并安装"股票监控" Skill，Skill 脚本在沙箱中运行，监测到价格触发条件时通过助理向用户发送通知和持仓分析卡片
