## 5. 非功能性需求

### 5.1 性能要求

| 场景 | 目标指标 |
|------|----------|
| 应用冷启动时间 | < 2 秒 |
| 首个 Token 响应时间（本地模型） | < 3 秒 |
| 首个 Token 响应时间（云端模型） | < 2 秒 |
| 实时语音对话延迟 | < 500ms（端对端） |
| 普通 STT 处理时间 | < 3 秒（30 秒音频） |
| 图片 OCR 处理时间 | < 1 秒 |
| UI 帧率 | 60 fps（普通）/ 120 fps（支持设备） |

### 5.2 兼容性要求

| 平台 | 最低系统版本 |
|------|-------------|
| iOS | iOS 16+ |
| Android | Android 10 (API 29)+ |
| macOS | macOS 13 (Ventura)+ |
| Windows | Windows 11 |

### 5.3 离线能力

- 核心对话功能在无网络环境下可用（需配置本地模型）
- 离线模式下自动禁用云端依赖功能并提示用户
- 本地 RAG 问答在无网络环境下完整可用

### 5.4 可访问性

- 支持系统级字体大小调整
- 支持 VoiceOver（iOS）/ TalkBack（Android）
- 高对比度模式支持
- 语音输入作为无障碍输入方式
