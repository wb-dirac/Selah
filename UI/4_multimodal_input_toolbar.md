## 4. 多模态输入工具栏

### 4.1 默认收起态

```
┌────────────────────────────────────────────┐
│  +   [  输入消息...              ] 🎤  📞  │
└────────────────────────────────────────────┘
  ↑                                      ↑
 展开工具                             主动作按钮
```

组件细节：
- 整体高度：52pt，带 Safe Area 兼容
- 输入框：多行自动扩展，最大展开 120pt，超出后内部滚动
- `+` 按钮：点击展开多模态工具栏（抽屉向上弹出）
- `🎤` 按钮：语音消息输入（点击或按住录音），用于发送语音消息，不进入全双工通话
- `📞` 主按钮：默认显示拨打图标，点击即进入音频全双工对话
- `⬆️` 发送：当输入框存在非空文本时，主按钮从 `📞` 切换为 `⬆️`，点击发送文本消息
- 输入框左侧：附件缩略图（已添加图片/文件时出现）

### 4.2 工具抽屉展开态

```
┌────────────────────────────────────────────┐
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │  📷  │  │  🖼️  │  │  📄  │  │  🎤  │  │
│  │ 拍照 │  │ 相册 │  │ 文件 │  │ 语音 │  │
│  └──────┘  └──────┘  └──────┘  └──────┘  │
│  ┌──────┐  ┌──────┐                       │
│  │  🔒  │  │  ···  │                       │
│  │本地  │  │更多  │                       │
│  └──────┘  └──────┘                       │
├────────────────────────────────────────────┤
│  +   [  输入消息...              ] 🎤  📞/⬆️ │
└────────────────────────────────────────────┘
```

各工具按钮尺寸：72×80pt，图标 32pt，标签 11px  
工具列表可横向滚动（当 Skill 注册了更多输入工具时）

**"本地处理"快捷切换按钮**（工具栏第一排末位）：
- 显示当前输入的处理策略（本地/云端）
- 点击弹出底部选择器，可为当前这条消息单独设置处理方式
- 颜色实时反映当前状态（绿=本地，蓝=云端）

### 4.3 图片/文件附件预览区

```
┌────────────────────────────────────────────┐
│  ┌─────┐ ┌─────┐                          │
│  │ 🖼️  │ │ 📄  │  +添加更多               │
│  │ ×  │ │ ×  │                          │  ← 附件预览行
│  └─────┘ └─────┘                          │
├────────────────────────────────────────────┤
│  [  补充说明这张图片...          ] 🎤  📞/⬆️ │
└────────────────────────────────────────────┘
```

图片缩略图：60×60pt，圆角 8px，右上角 × 删除按钮  
文件缩略图：显示文件类型图标 + 文件名截断 + 文件大小  
添加附件后：输入框 placeholder 变为"补充说明..."

### 4.4 语音消息输入态（点击/按住录音）

```
┌────────────────────────────────────────────┐
│                                            │
│              🎤                            │
│        松开发送 · 上滑取消                   │
│                                            │
│  ████████████░░░░░░░  [实时波形动画]        │
│                                            │
│     [录音时长 00:08] [音量波形]              │
│   松开发送语音消息 / 上滑取消                │
└────────────────────────────────────────────┘
```

行为规范：
- 点击或按住麦克风后：背景暗化，聚焦到录音界面
- 实时波形：白色线条，随音量动态变化
- 上滑超过 80pt：进入"取消"状态，红色警告色，松开不发送
- 松开（正常区域）：直接发送语音消息（二进制音频）到多模态 LLM
- 发送成功后：在消息列表展示语音气泡（时长/播放控件）
- 若本轮输入是语音消息：AI 文字回复完成后自动触发 TTS 播放（按用户所选供应商）
