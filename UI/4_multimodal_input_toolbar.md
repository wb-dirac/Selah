## 4. 多模态输入工具栏

### 4.1 默认收起态

```
┌────────────────────────────────────────────┐
│  +   [  输入消息...              ] 🎤  ⬆️  │
└────────────────────────────────────────────┘
  ↑                                      ↑
 展开工具                             发送/语音
```

组件细节：
- 整体高度：52pt，带 Safe Area 兼容
- 输入框：多行自动扩展，最大展开 120pt，超出后内部滚动
- `+` 按钮：点击展开多模态工具栏（抽屉向上弹出）
- `🎤` 按钮：按住说话（普通语音模式）；长按 1.5s 切换至实时对话模式
- `⬆️` 发送：输入内容非空时激活为主色调；空态为灰色禁用
- 输入框左侧：附件缩略图（已添加图片/文件时出现）

### 4.2 工具抽屉展开态

```
┌────────────────────────────────────────────┐
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │  📷  │  │  🖼️  │  │  📄  │  │  🎤  │  │
│  │ 拍照 │  │ 相册 │  │ 文件 │  │ 语音 │  │
│  └──────┘  └──────┘  └──────┘  └──────┘  │
│  ┌──────┐  ┌──────┐                       │
│  │  🔒  │  │  ···  │                       │
│  │本地  │  │更多  │                       │
│  └──────┘  └──────┘                       │
├────────────────────────────────────────────┤
│  +   [  输入消息...              ] 🎤  ⬆️  │
└────────────────────────────────────────────┘
```

各工具按钮尺寸：72×80pt，图标 32pt，标签 11px  
工具列表可横向滚动（当 Skill 注册了更多输入工具时）

**"本地处理"快捷切换按钮**（工具栏第一排末位）：
- 显示当前输入的处理策略（本地/云端）
- 点击弹出底部选择器，可为当前这条消息单独设置处理方式
- 颜色实时反映当前状态（绿=本地，蓝=云端）

### 4.3 图片/文件附件预览区

```
┌────────────────────────────────────────────┐
│  ┌─────┐ ┌─────┐                          │
│  │ 🖼️  │ │ 📄  │  +添加更多               │
│  │ ×  │ │ ×  │                          │  ← 附件预览行
│  └─────┘ └─────┘                          │
├────────────────────────────────────────────┤
│  [  补充说明这张图片...          ] 🎤  ⬆️  │
└────────────────────────────────────────────┘
```

图片缩略图：60×60pt，圆角 8px，右上角 × 删除按钮  
文件缩略图：显示文件类型图标 + 文件名截断 + 文件大小  
添加附件后：输入框 placeholder 变为"补充说明..."

### 4.4 语音输入态（按住说话）

```
┌────────────────────────────────────────────┐
│                                            │
│              🎤                            │
│        松开发送 · 上滑取消                   │
│                                            │
│  ████████████░░░░░░░  [实时波形动画]        │
│                                            │
│  "帮我查一下..."  [实时 STT 文字显示]        │
│         🔒 本地识别中                       │
└────────────────────────────────────────────┘
```

行为规范：
- 按下后：背景暗化，聚焦到录音界面
- 实时波形：白色线条，随音量动态变化
- 实时 STT：转写结果实时显示在底部，让用户确认识别准确性
- 上滑超过 80pt：进入"取消"状态，红色警告色，松开不发送
- 松开（正常区域）：发送语音转文字结果
- 识别标记：底部常驻"本地识别中 🔒"或"云端识别中 ☁️"
