## 5. 实时语音对话界面

### 5.1 全屏沉浸态

```
┌─────────────────────────────────────────┐
│                                  ✕ 结束  │  ← 仅保留退出按钮
│                                         │
│                                         │
│          ╔═══════════════╗              │
│          ║               ║              │
│          ║   [声波动画]   ║              │  ← 中央主视觉
│          ║               ║              │
│          ╚═══════════════╝              │
│                                         │
│      正在聆听...                         │  ← 状态文字
│                                         │
│  ┌─────────────────────────────────┐   │
│  │  [实时字幕区域]                  │   │  ← 字幕区，半透明背景
│  │  "好的，我来帮你查一下..."       │   │
│  └─────────────────────────────────┘   │
│                                         │
│     🔇静音      ⏸️暂停      ☁️云端       │  ← 底部控制栏
└─────────────────────────────────────────┘
```

背景设计：深色渐变背景（非纯黑，保持温度），配合微弱的粒子动画表现"思考中"。

**声波动画状态规范**：

| 状态 | 动画描述 | 颜色 |
|------|----------|------|
| 待机/静默 | 平静低振幅圆形波纹 | 灰蓝色 |
| 用户说话 | 高振幅随声音实时波动 | 白色主波 + 蓝色外环 |
| AI 处理中 | 缓慢旋转的粒子环，表示"思考" | 主色调蓝 |
| AI 说话 | 平滑的输出波形，有节奏感 | 主色调蓝，较饱和 |
| 打断（Barge-in） | 波形快速收缩然后切换为用户波形 | 动画过渡流畅 |

**字幕区设计**：
- 半透明黑色背景（rgba(0,0,0,0.6)），毛玻璃效果
- 当前 AI 说话内容用白色显示，字号 16px
- 用户刚说的话用灰色显示，字号 14px
- 字幕自动滚动，始终显示最新内容
- 可点击字幕区收起（降低干扰）

**底部控制栏**：
- 静音：麦克风 icon，激活时显示红色划线
- 暂停：暂停整个对话，转为普通文字模式
- 云端标记：不可交互，仅展示当前使用的服务名称
