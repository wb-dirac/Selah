```
┌─────────────────────────────────────────┐
│  ──                                     │  ← 下拉指示条
│  添加提供商                              │
├─────────────────────────────────────────┤
│  选择类型                               │
│                                         │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐   │
│  │[A] │ │[O] │ │[G] │ │[L] │ │[自] │  │  ← 快速选择图标
│  │ An │ │ OAI│ │ GL │ │本地│ │定义│  │
│  └────┘ └────┘ └────┘ └────┘ └────┘   │
│                                         │
├─────────────────────────────────────────┤
│  Anthropic                              │
│  ─────────────                          │
│  API Key                                │
│  ┌─────────────────────────────────┐   │
│  │ sk-ant-api03-···············    │   │  ← 密文展示，右侧眼睛图标
│  └─────────────────────────────────┘   │
│  ⚠️ Key 将加密存储在本地 KeyChain，不上云 │  ← 安全提示
│                                         │
│  别名（可选）                            │
│  ┌─────────────────────────────────┐   │
│  │ 我的 Claude Key                  │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [测试连接]              [保存]          │
└─────────────────────────────────────────┘
```

"测试连接"状态反馈：
- 点击后按钮显示 loading spinner
- 成功：按钮变绿色 "✓ 连接成功"，持续 2 秒后恢复
- 失败：按钮变红色 "✗ 连接失败"，下方展示错误信息
