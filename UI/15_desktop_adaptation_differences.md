## 15. 桌面端适配差异

### 15.1 macOS / Windows 布局差异

**侧边栏**：取代移动端底部 Tab Bar，宽度 260px，可折叠为 60px 图标模式。

**分栏布局**：屏幕宽度 > 1024px 时，自动切换为左侧对话列表 + 右侧对话详情的双栏视图。

**键盘快捷键**：
```
⌘N / Ctrl+N     新建对话
⌘K / Ctrl+K     全局搜索
⌘, / Ctrl+,     打开设置
⌘⏎ / Ctrl+⏎    发送消息（而非普通回车换行）
⌘/ / Ctrl+/     打开命令面板
Esc             关闭弹窗 / 取消当前操作
```

**拖拽上传**：支持直接将文件拖拽到对话窗口中上传

**右键菜单**：消息气泡右键 → 复制/重试/分支/引用回复

**窗口管理**：支持多窗口（每个对话独立窗口）

### 15.2 桌面端独有界面元素

**命令面板**（⌘K 唤起）：
```
┌─────────────────────────────────────────┐
│  🔍  输入命令或搜索...                  │
├─────────────────────────────────────────┤
│  最近使用                               │
│  💬  新建对话                           │
│  ⚙️  打开设置                           │
│  🔌  切换模型 → Claude Sonnet 4.6       │
│  📋  查看定时任务                        │
└─────────────────────────────────────────┘
```

**菜单栏（macOS）**：
- 应用菜单：关于、偏好设置、隐私...
- 文件：新建对话、导出...
- 编辑：搜索、清除上下文...
- 视图：侧边栏、紧凑模式...
