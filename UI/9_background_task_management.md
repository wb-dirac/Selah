```
┌─────────────────────────────────────────┐
│  ← 取消          新建任务       保存     │
├─────────────────────────────────────────┤
│  任务名称                               │

│  │ 到时间提醒我给张总回邮件，如果我      ││
│  │ 没回复就帮我起草一封回复...           ││

│                                         │

│  ○  指定时间    ● 固定时间              │  ← 单选，多选见下方

│  ○  条件触发                            │
│                                         │
│  ─── 固定时间配置 ───                   │
```
