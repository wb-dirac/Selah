## 6. 生成式 UI 卡片组件库

### 6.1 卡片通用设计规范

所有生成式 UI 卡片共享以下基础样式：

```
背景: --color-bg-surface
边框: 1px solid --color-border
圆角: --radius-xl (16px)
阴影: 层级 1
内边距: 16px
最大宽度: 对话流宽度的 100%
卡片顶部: 类型标签 + 来源说明（小字，灰色）
卡片底部: 操作按钮区（如有）
```

卡片类型标签示例：
```
[🚂 高铁信息]   [🛍️ 商品推荐]   [📍 地图预览]   [📅 日历事件]
```

### 6.2 高铁/火车票卡片 (`train_card`)

```
┌─────────────────────────────────────────┐
│  🚂 高铁信息                 G1次 · 二等座│
├─────────────────────────────────────────┤
│                                         │
│   北京南          →        上海虹桥      │
│   08:05                   12:28         │
│   ───────────────────────────           │
│              4小时23分 · ¥553           │
│                                         │
├─────────────────────────────────────────┤
│  📅 明天 周二  3月3日                    │
│  💺 余票充足                             │
├─────────────────────────────────────────┤
│  [加入日历]          [前往购票]           │
└─────────────────────────────────────────┘
```

交互细节：
- "加入日历"：调用 Tool Bridge 日历工具（需 L2 确认）
- "前往购票"：URL Scheme 唤起 12306 App 或打开网页
- 长按卡片：显示"分享"、"复制信息"操作

### 6.9 代码块组件 (`code_block`)

```
┌─────────────────────────────────────────┐
│  python                      [复制] [运行]│  ← 语言标签 + 操作
├─────────────────────────────────────────┤
│  import pdfplumber             ··       │
│                                         │
│  with pdfplumber.open("doc.pdf") as p:  │
│      text = p.pages[0].extract_text()   │
│      print(text)                        │
└─────────────────────────────────────────┘
```

- 背景：#1E1E2E（深色代码主题，独立于 light/dark mode）
- 语法高亮：关键字蓝色，字符串绿色，注释灰色
- "运行"按钮：仅当 Skill 沙箱可执行该语言时显示，点击在沙箱中运行并显示输出
