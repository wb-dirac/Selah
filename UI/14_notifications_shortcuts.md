## 14. 通知与快捷操作

### 14.1 定时任务触发通知

```
┌─────────────────────────────────────────┐
│  🤖 助理提醒                   14:00    │
│  ─────────────────────────────────────  │
│  你有 15 分钟前还没有回复张总的邮件       │
│  要现在回复吗？                          │
│                                         │
│  [帮我起草]        [稍后提醒]   [忽略]  │  ← 可交互通知
└─────────────────────────────────────────┘
```

点击"帮我起草"：直接打开 App 并进入邮件起草对话；
点击"稍后提醒"：弹出时间选择器（15分钟/1小时/明天）

### 14.2 工具调用确认通知（后台任务）

```
┌─────────────────────────────────────────┐
│  🤖 助理需要你的确认                14:01│
│  ─────────────────────────────────────  │
│  即将发送邮件给 zhangsan@example.com     │
│  主题: Re: 项目进展汇报                  │
│                                         │
│  [查看草稿]               [确认发送]    │
└─────────────────────────────────────────┘
```

### 14.3 Toast 消息规范

```
成功 Toast（绿色，底部居中，2秒后自动消失）：
  ┌──────────────────────┐
  │ ✓  已添加到日历       │
  └──────────────────────┘

错误 Toast（红色，底部居中，4秒，带关闭按钮）：
  ┌─────────────────────────────┐
  │ ✗  网络请求失败，请稍后重试  ×│
  └─────────────────────────────┘

进行中 Toast（蓝色，不自动消失）：
  ┌───────────────────────┐
  │ ◌  正在连接 Agent...   │
  └───────────────────────┘
```
