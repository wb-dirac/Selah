## 2. 导航结构与信息架构

### 2.1 移动端底部 Tab Bar

```
┌─────────────────────────────────────────┐
│                                         │
│           [主内容区域]                   │
│                                         │
├────────┬────────┬────────┬──────────────┤
│  对话   │  任务   │  Agent │   设置       │
│  💬     │  📋     │  🔗    │   ⚙️        │
│ Chat   │ Tasks  │ Agents │ Settings    │
└────────┴────────┴────────┴─────────────┘
```

**Tab 详细说明**：

| Tab | 图标 | 对应模块 | Badge 规则 |
|-----|------|----------|------------|
| 对话 | 气泡图标 | 对话历史列表 + 当前对话 | 无未读时无 badge |
| 任务 | 清单图标 | 定时任务列表 | 有待触发任务时显示数量 |
| Agent | 节点连接图标 | A2A 已连接 Agent 列表 | 有新发现的 Agent 时显示圆点 |
| 设置 | 齿轮图标 | 全局设置中心 | 有重要配置未完成时显示警告点 |

Tab Bar 视觉细节：
- 选中态：图标填充色 + 文字加粗，底部 3px 高亮线（--color-primary）
- 未选中态：图标描边风格，文字 --color-text-muted
- 背景：毛玻璃效果 (backdrop-filter: blur(20px))，保持与内容的视觉连续性
- 高度：iPhone 54pt + Safe Area；Android 56dp + Navigation Bar
